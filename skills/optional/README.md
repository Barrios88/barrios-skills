# Optional skills

These skills are **not part of the core Barrios collection** but remain available if you need them.

## Included here

- **Documents** — Word, PDF, Excel, PowerPoint, markitdown
- **Career** — recommendation letters, CVs, research statements
- **Specialty data tools** — aeon, networkx, shap, vaex, umap, sympy, simpy
- **Extra research/writing** — paper-2-web, scientific schematics, Notion docs, consulting-style reports

Install the same way as core skills — copy the folder to `~/.cursor/skills/`, `~/.claude/skills/`, or `~/.codex/skills/`.
