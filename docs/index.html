Two kinds of tools in this repo — skills (markdown instructions the agent reads) and MCP servers (separate programs that connect to data sources like WRDS). They work together but install differently.

Browse skills MCP servers

Skills · skills/

Copy a folder to ~/.cursor/skills/. Teaches Stata, LaTeX, writing voice, literature search.

MCP servers · mcp/

Install once, register in Cursor/Claude, add your credentials. Gives the agent live WRDS query tools.

# Skill (copy markdown instructions)
cp -R skills/econometrics/stata-regression \
     ~/.cursor/skills/

# MCP (separate — see mcp/ folder)
pip install -e mcp/wrds-mcp
# + your WRDS_USERNAME / WRDS_PASSWORD

MCP servers · separate from skills

Connect to live data sources

MCP servers live in mcp/, not skills/. They are Python programs registered in Cursor or Claude — you install them once and provide your own account credentials (e.g. WRDS username and password).

MCP server · not a skill

wrds-mcp →

Query Compustat, CRSP, Form 4, TRACE, and other WRDS libraries via MCP tools like wrds_run_sql. Requires your WRDS account, Duo MFA, and a one-time tunnel setup per session.

Full setup guide — credentials, tunnel, Cursor & Claude config
