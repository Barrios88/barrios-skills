MCP servers · live data connections

WRDS MCP

Full setup guide

Install from mcp/wrds-mcp/. Requires your WRDS account, Duo MFA, and MCP registration in Cursor or Claude. Pair with the wrds skill for query filters.

  • Runs queries — the agent calls tools like wrds_run_sql, not copy-paste SQL
  • Your credentials — each user connects with their own WRDS account
  • One Duo per session — the tunnel reuses a single approved connection
  • Pairs with the wrds skill — MCP executes; the skill enforces Compustat and CRSP filters

Featured MCP server

wrds-mcp

Query Compustat, CRSP, Form 4, TRACE, TAQ, and other WRDS libraries through MCP tools. One Duo approval per session via the persistent tunnel.

Install

Get started in three steps

Skills and MCP install differently — pick the path you need.

1

Clone the repo

git clone https://github.com/Barrios88/barrios-skills.git
2

Unzip and copy the skill

unzip stata-regression.zip
cp -R stata-regression ~/.cursor/skills/
3

Ask naturally

The agent loads the skill when your request matches its description.

Cursor

~/.cursor/skills/

Claude Code

~/.claude/skills/

Codex

~/.codex/skills/

Use your own WRDS username and password throughout. See the detailed guide for troubleshooting.

1

Install the server

cd barrios-skills/mcp/wrds-mcp
python3 -m venv ~/.wrds-mcp-env
source ~/.wrds-mcp-env/bin/activate
pip install -e .
2

Set your credentials & tunnel

export WRDS_USERNAME="your_wrds_username"
export WRDS_PASSWORD="your_wrds_password"
bash mcp/wrds-mcp/tunnel/tunnel_up.sh
3

Register in Cursor or Claude

# Add WRDS_USERNAME, WRDS_PASSWORD,
# WRDS_TUNNEL_PORT=49600 to MCP config

~/.cursor/mcp.json
