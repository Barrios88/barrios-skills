# Upstream MCP notes

This folder documents MCP servers that **Barrios Skills** recommends but does not vendor.

| Server | Location |
|--------|----------|
| **WRDS MCP** (vendored) | [`wrds-mcp/`](wrds-mcp/) — John Barrios |
| **SEC EDGAR, OpenEcon, FRED, Stata-MCP** | See [`../install/external-mcps.md`](../install/external-mcps.md) |

Skills teach *how* to work. MCP servers give the agent *live tools*. Pair them:

- `wrds` skill ↔ wrds-mcp  
- `sec-edgar` skill ↔ SEC EDGAR MCP  
- `api-data-fetcher` skill ↔ OpenEcon Data / FRED MCP  
- `stata-regression` skill ↔ Stata-MCP
